# Архитектура Transformer для русско-английского машинного перевода

Данный проект представляет собой реализацию архитектуры **Transformer** на PyTorch. Модель обучена на датасете **OPUS-100** для выполнения задачи перевода с **русского языка на английский**. 

---

## Что реализовано

  - Реализованы все ключевые компоненты: `MultiHeadAttention`, `PositionalEncoding`, `Encoder/Decoder Layers`, `FeedForward` и `LayerNorm`.
  - Полное повторение архитектуры из статьи ["Attention is All You Need"](https://arxiv.org/abs/1706.03762).
  - Обучен BPE-токенизатор с размером словаря 32,000 токенов, адаптированный под русский и английский языки.
  - Фильтрация данных (регулярные выражения) для удаления артефактов (китайские иероглифы, спецсимволы).
  - Использование **Gradient Accumulation** для симуляции большого размера батча (эффективный batch size = 32) при ограниченной видеопамяти.
  - Автоматическое сохранение чекпоинтов (Checkpoints) для возобновления обучения.

## Архитектура 

| Компонент | Значение |
| :--- | :--- |
| Количество слоев (Layers) | `6` (Encoder & Decoder) |
| Размерность модели (`d_model`) | `512` |
| Количество голов внимания (`heads`) | `8` |
| Размерность FFN (`d_ff`) | `2048` |
| Максимальная длина последовательности | `200` токенов (обучение), `100` (генерация) |
| Размер словаря (BPE) | `32,000` + спецтокены (`[PAD]`, `[UNK]`, `[BOS]`, `[EOS]`) |

## Как это работает 

1.  **Токенизация:** Текст разбивается на подслова (BPE), чтобы эффективно обрабатывать редкие слова русского языка.
2.  **Кодирование:** Энкодер читает русское предложение, используя Self-Attention, чтобы понять контекст и связи между словами.
3.  **Декодирование:** Декодер генерирует перевод слово за словом. На каждом шаге он использует:
    *   **Masked Self-Attention** — чтобы "смотреть" только на уже переведенные слова.
    *   **Cross-Attention** — чтобы обращаться к закодированному русскому предложению для выбора правильного смысла.
4.  **Выход:** Слой `Linear` и `Softmax` превращают выход декодера в вероятности для каждого слова английского словаря.
