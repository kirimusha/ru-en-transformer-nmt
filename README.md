# Архитектура Transformer для русско-английского машинного перевода

Модель обучена на датасете OPUS-100, на ru-en паре.

Веса модели: https://drive.google.com/drive/folders/1myrFmVeuuM07lNLOnLUPWoY--3BmXane?usp=sharing

## Параметры модели

| Компонент | Значение |
| :--- | :--- |
| Количество слоёв | 6 (кодер и декодер) |
| Размерность модели (d_model) | 512 |
| Количество голов внимания | 8 |
| Размерность полносвязного слоя (d_ff) | 2048 |
| Максимальная длина последовательности | 200 токенов (обучение), 100 (генерация) |
| Размер словаря (BPE) | 32 000 + служебные токены |

## Результаты

| Метрика | Значение |
| :--- | :--- |
| BLEU | 20.01 |
| METEOR | 0.3927 |
| TER | 1.0747 |

Полученный показатель BLEU равен 20.01 при эталонном диапазоне от 0 до 100; значения выше 20 считаются минимально достаточными для сохранения общего смысла предложений. Метрика METEOR составила 0.3927 при эталонном интервале от 0 до 1, где диапазон 0.3–0.5 соответствует среднему качеству перевода с передачей ключевой семантики. По метрике TER оптимальным результатом является стремление к 0, а нормой для качественного перевода — диапазон 0.4–0.6. Итоговый показатель TER составил 1.0747. Превышение единицы обусловлено ошибкой генерации — избыточным циклическим повтором токенов.
